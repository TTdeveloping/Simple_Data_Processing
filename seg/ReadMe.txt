这是一个数据对法律数据中的案情描述部分(即fact部分)进行jieba分词的程序：
legal_data.json是原始数据，即要分词的数据
seg_legal_data.json是案情描述(fact)部分被jieba分词之后的数据

This is a procedure for data jieba word segmentation in the case description part of the legal data (ie the fact part):
Legal_data.json is the original data, that is, the data to be segmented
Seg_legal_data.json is the data after the fact part of the case is divided by jieba
Legal_data_segment.py is the running code
